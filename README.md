# ChemLearn

Mã nguồn cho dự án ChemLearn – nền tảng học và ôn tập Hóa học dành cho môn CT275 Công nghệ Web (Đại học Cần Thơ).

## Cấu trúc thư mục
Xem chi tiết trong thư mục [`chemlearn/`](chemlearn/), bao gồm các thư mục MVC, tài nguyên tĩnh và tệp cấu hình.

## Bắt đầu nhanh
1. Sao chép thư mục `chemlearn` vào máy chủ PHP (ví dụ: `htdocs` của XAMPP).
2. Tạo cơ sở dữ liệu MySQL bằng file [`chemlearn/chemlearn.sql`](chemlearn/chemlearn.sql).
3. Cập nhật thông tin kết nối trong [`chemlearn/config/config.php`](chemlearn/config/config.php).
4. Truy cập `http://localhost/chemlearn/` để trải nghiệm website.

Chi tiết hướng dẫn và mô tả dự án được trình bày trong [`chemlearn/README.md`](chemlearn/README.md).
